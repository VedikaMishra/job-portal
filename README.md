# Job Portal System

A mini job portal system built using *Flask, **Python, and **MySQL* for learning and practice purposes.

## 🔧 Features
- User registration and login
- Job posting
- Job searching and applying
- Admin panel for managing jobs

## 🛠️ Technologies Used
- Python
- Flask
- MySQL
- HTML
- CSS
